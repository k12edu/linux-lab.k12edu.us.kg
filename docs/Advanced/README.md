# 進階開發技巧

## 說明
<span style="font-size: 20px;">本教程適合已熟悉程式設計、軟體開發及伺服器相關技術，並完成中階開發技巧學習的學員。
本教程專為想要更深入學習 IT 開發技巧的學員，範圍較窄，但複雜度更高，目前包含 CI/CD 自動化流程和 Kubernetes 等實用技能。  
建議學員在開始前明確自身需求與學習目標，並善用教程中提供的資源和指引，逐步提升自身的開發能力。
